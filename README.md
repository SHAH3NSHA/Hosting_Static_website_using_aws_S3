# 2310030411_lab

Static Website Hosting using AWS S3 & Render (Free Deployment)

Project Overview

This project demonstrates how to host a static website using cloud platforms.
The website is deployed using:

* AWS S3 for static website hosting
* Render for free public hosting

The goal of this project is to understand cloud storage, hosting, and deployment of web applications.

---

Technologies Used

* HTML (for webpage structure)
* AWS S3 (for hosting static website)
* Render (for free deployment)
* GitHub (for version control and code hosting)

---

 Project Structure

```
my-website/
│
├── index.html
```

---

Deployment Methods

 1. AWS S3 Hosting

The website is hosted on AWS S3 by:

* Creating an S3 bucket
* Uploading website files
* Enabling static website hosting
* Setting bucket permissions to public

This generates a public S3 endpoint to access the website.

---

2. Render Hosting (Free)

The website is also deployed using Render by:

* Uploading code to GitHub
* Connecting the repository to Render
* Deploying as a static site

Render provides a free `.onrender.com` URL with HTTPS support.
